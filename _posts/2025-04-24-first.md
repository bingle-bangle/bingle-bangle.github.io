1. 프로젝트 개요
게임 제목: 비트 히어로 (Bit Hero)

장르: 텍스트 기반 선택형 어드벤처

플랫폼: 콘솔 (C++로 구현) -> 언리얼로 구현 예정

개발 목적:

C++ 기초 문법을 실습하고, 향후 배울 C++ 개념들을 지속적으로 코드에 통합 및 확장할 것입니다.
현재 실습한 개념 : 변수, 상수, 연산자, 2진수/16진수 변환, 비트 연산, 난수 함수, 열거형, 조건문, 반복문

2. 스토리 및 콘셉트
시놉시스:
왕국의 공주가 마왕에게 납치되었다!
플레이어는 '비트 히어로'가 되어 마왕성에 입성해, 다양한 함정과 몬스터을 물리쳐 공주를 구해야 한다.
이동에 따라 이벤트가 발생하고, 성공적으로 마왕을 물리치면 공주를 구할 수 있다.

3. 현재 구현된 기능
   -이동 시스템: 걷기/달리기/점프/공격 명령 선택
   -함정 등장: 이동 시 일정 확률로 함정 등장 및 확률에 따라 회피
   -퍼즐 문제: 비트연산 문제,쉬프트 연산 문제,2진수 ↔ 16진수 변환 문제


4. 앞으로 추가하거나 수정할 계획
🔵 게임 기능 추가 계획
  1) HP, 스태미나, 거리 표시 HUD 강화
      → 현재 상태를 매 턴마다 출력해서 몰입감 증가
  2) 보스전 구현
      → 최종 보스 (마왕)와 마지막 전투
  3) 아이템 시스템
      → 체력 회복 아이템, 함정 무시 아이템 등 (추후 난수 등장)
  4) 세이브/로드 기능 (나중에 파일 입출력 배운 후)
      -> 2~3개의 엔딩 추가해 특정 시점 로드 기능 구현 예정

🛠 코드 수정 및 보완 계획
    1) 코드 구조화
        → 지금은 main()에 모든 로직이 몰려있어 가독성이 매우 떨어짐으로 앞으로 배울 개념을 추가해 코드를 단순하고 가독성을 높일 예정
    2) 입력 유효성 강화
        → 잘못된 입력 처리, 숫자가 아닌 값 입력 시 예외 처리 추가
    3) 랜덤 이벤트 다양화
        → 함정 문제를 랜덤하게 처리할 예정(비트 연산 등), 특정 아이템 랜덤 등장


5. 개발 상황
